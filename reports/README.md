# NarrativeIQ Reports

Reports use the full logo asset generated by:

```powershell
python scripts/create_logo_assets.py
```

Frontend report previews and PDF exports use `reports/assets/narrativeiq-full.png`.

## Generate Exhibition Report

```powershell
python reports/generate_report.py
```

The script reads `datasets/marts/narrativeiq_mart.json` and writes:

```text
reports/generated/narrativeiq_exhibition_report.html
```

Open the HTML file in a browser or print it to PDF for the project exhibition.

## Export PDF

```powershell
python reports/export_report_pdf.py
```

The exporter uses installed Chrome or Microsoft Edge in headless mode and writes:

```text
reports/generated/narrativeiq_exhibition_report.pdf
```
